# Project Title

This is a 2D arcade style shooting game inspired by the Robotron 2084 game

## How to play

-  Use the AWSD or arrow keys to control the character's movement and the cursor to control where to shooting
-  Destroy all enemies and you progress to the next level
-  Levels 1-6 are static, and level 7 and beyond are dynamically generated

## Installation

-  Clone the Repository
-  compile with javac GameDriver.java
-  run with java GameDriver
